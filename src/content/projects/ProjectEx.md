# VFX Power-Ups in Unity

Academic real-time graphics project focused on a third-person character that interacts with three power-up objects.

## Core Features

- Third-person player controller (Starter Assets base).
- Three interactive power-ups.
- Custom visual state per power-up using shaders.
- Particle systems synchronized with gameplay interaction.

## Technical Stack

- Unity 2022.3 LTS
- URP (Universal Render Pipeline)
- C#, ShaderLab, HLSL

## Goal

Demonstrate practical use of visual effects workflows by combining shaders, particles, and interaction systems in real time.
